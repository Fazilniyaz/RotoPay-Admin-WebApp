🔗 API INTEGRATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

To connect your backend:

1. Set environment variable:
   NEXT_PUBLIC_API_URL=http://your-backend.com

2. Implement backend endpoints (see DEPLOYMENT.md)

3. Replace mock data with API calls using:
   import api from '@/lib/axios';
   const response = await api.get('/api/endpoint');

4. Deploy to Vercel

💡 PRO TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• ALWAYS use centralized axios instance (lib/axios.ts)
  Never use fetch() or create separate axios instances!

• Check ARCHITECTURE.md for data models and API flows

• Customize colors in app/globals.css CSS variables

• Add new pages in app/dashboard/new-page/page.tsx

• Update navigation in components/layout/DesktopSidebar.tsx
  AND components/layout/MobileBottomNav.tsx

• Test responsive design: Press F12 → Device emulation

• Check TypeScript: pnpm tsc --noEmit
